### How Connections Work

**Claude Account:**
- Click "Connect" in dashboard
- One-click connection (simulated)
- Enables better AI briefs

**Google Calendar:**
- Click "Connect" in dashboard
- Loads upcoming meetings
- Click any meeting to generate brief

**All data stored locally - no backend needed!**
